Para comenzar vamos a conocer algunas medidas de css.

Medidas Relativas

Las medidas relativas están conformadas por
  • px
    • px hace referencia a los pixeles que quieren utilizarse
    para esta medida, puede ser aplicado a valores como width,
    height, font-size, margin, padding, etc
    un ejemplos sería este:

    texto

    al cual le aplicamos 30px de font-size.
  • em
    • em es relativa a el tamaño de letra actual, por ejemplo, si tenemos un:

    Texto y dentro de este otro

    texto dentro

    en el primer texto tenemos un tamaño de 20px y en el siguiente texto tenemos un tamaño de 2em lo cual es el doble del primer texto que lo contenía.

  • ex
    • ex depende del tamaño de la fuente que se está utilizando tomando como base su tamaño inicial y parte de ahí para añadir-quitar tamaño

    Este es un texto de ejemplo

    A este texto le estamos dando un 3 de ex
  • rem
    • rem hace referencia al font-size que ya hemos definido dentro de una etiqueta

    Este es un texto de ejemplo aqui le damos font-size de 15px

    aqui le damos tamaño de 1.5rem

Medidas Absolutas

  • in

    • Indica cuantas pulgadas de medida aqui tiene 0.1in

  • cm

    • indica cuantos cm de medida, aqui tiene .5cm

  • pt

    • indica cuantos puntos (unidad de medida de tipografías) se estan utilizando aqui son 20pt;

  • mm

    • indica cuantos milímetros de tamaño, aquí son 12mm

  • pc

    • indica cuantas picas (unidad de medida igual a 12pt de fuente)este texto tiene 1.1pc

Las medidas Porcentuales se utilizan con % indican el tamaño en relación a el contenedor en el que se encuentran: por ejemplo.

en este texto estoy manejando una medida de 10px

en este texto tiene una medida de 200%
